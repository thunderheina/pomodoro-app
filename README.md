### Pomodoro
------------
Сверстано по figma-макету. Верстка адаптивная. При верстке был использован React, препроцессор Sass, иконки импортированы через React.

#### Запуск проекта:
Установка зависимостей:

`npm install`

Запуск проекта в режиме разработки:

`npm start`

Сборка проекта:

`npm run build`
